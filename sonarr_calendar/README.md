# Sonarr Calendar Tracker

A beautiful HTML dashboard for your Sonarr shows, showing upcoming episodes, progress bars, and season completion.

## Features
- 📅 Configurable date range
- 🖼️ Poster image caching
- 🎨 Dark/light theme toggle
- 📊 Overall and per‑show progress
- 🏆 Recently completed seasons section
- 🔄 Auto‑refresh mode
- ⌨️ Graceful Ctrl+C handling
- 🌍 OS‑aware date formatting
